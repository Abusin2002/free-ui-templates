This repository contains multiple reusable frontend templates, dashboards, and UI components designed for developers, portfolios, and business applications.

---

## ▶️ Running a Template Locally

Navigate into any project folder:

```bash
cd folder-name
```

Example:

```bash
cd goldnest-admin
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Open the local URL shown in terminal.
